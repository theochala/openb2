Proxies that you import should have one of the following formats:
Syntax Examples
host:port 127.0.0.1:8080, myproxy.com:1234
host:port:username:password 127.0.0.1:8080:myusername:mypassword
(type)host:port (http)127.0.0.1:8080
(type)host:port:username:password (socks5)myproxy.com:5134:myusername:mypassword
The proxy type can be either http, socks4, socks4a or socks5.
