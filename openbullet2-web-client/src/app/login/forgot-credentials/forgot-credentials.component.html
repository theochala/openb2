If you're the admin

Edit your UserData/OpenBulletSettings.json file with any text editor, and in the SecuritySettings section you will find AdminUsername and AdminPasswordHash. Replace the password hash with a new BCrypt hash, generated with a tool like this one.

Then restart the application and login with the new password.

If you're a guest

There's nothing you can do, you will have to ask the admin to reset your password for you.
